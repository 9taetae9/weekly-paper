## 🇶 Spring Framework가 탄생하게 된 배경과 이를 통해 해결하고자 했던 문제점에 대해 설명하세요.
- Spring Framwork 탄생 이전에 기업들은 Java의 표준 기술로 EJB(Enterprise Java Beans)를 사용하였음
### EJB의 문제점
- 객체 지향적이지 않음
- 높은 컨터이너 의존성으로 테스트가 쉽지 않음
- 프로그래밍 모델이 복잡함
